# SqlPulse
SqlPulse Pro is an SSMS (SQL Server Management Studio) extension
  for SQL Server developers and DBAs.

  Features included in Pro:
  • Inline result grid filtering – filter query results without
    re-running queries
  • Query history – automatic capture and replay of executed queries
  • Session restore – reopen your tabs exactly where you left off
  • Tab customization – rename and color-code editor tabs
  • DB Groups – organize Object Explorer connections by color and rules
  • SQL Formatter – format SQL with configurable style rules
  • Export to CSV/Excel – save result grids directly from SSMS

  Works with SSMS 22 (SQL Server Management Studio).
  Requires Windows, .NET Framework 4.8.

  License is per-computer. Activation requires internet connection
  once; works fully offline after that.
